Solicitação de Exclusão de Dados

Esta página explica como solicitar a exclusão de alguns ou todos os dados do usuário sem exigir a exclusão completa da conta no aplicativo Territórios.

1. Etapas para solicitar a exclusão de dados

  1. Entre em contato com o administrador ou responsável local pelo aplicativo.
  2. Informe o e-mail usado para entrar no sistema.
  3. Descreva quais dados você deseja excluir ou corrigir, de forma específica.
  4. Se necessário, confirme a titularidade da conta para proteção contra exclusões indevidas.

2. Tipos de dados que podem ser excluídos

  • Dados de perfil, como nome exibido e número de WhatsApp.
  • Token de notificações push vinculado ao usuário.
  • Dados acessórios que não precisem ser preservados para segurança ou histórico operacional.

3. Tipos de dados que podem ser mantidos

Alguns dados podem ser mantidos quando forem necessários para segurança, integridade do sistema, prevenção a fraude, auditoria, cumprimento de obrigação aplicável ou preservação de histórico administrativo e operacional.

  • Registros mínimos de segurança e logs técnicos.
  • Histórico administrativo e operacional já consolidado no sistema.
  • Informações cuja retenção seja necessária por obrigação legítima aplicável.

4. Período adicional de armazenamento

Quando houver necessidade de retenção adicional, registros mínimos de segurança e dados operacionais estritamente necessários podem ser mantidos por até 90 dias. Se existir obrigação legal, necessidade de auditoria ou justificativa legítima de segurança, esse prazo poderá ser maior pelo tempo estritamente necessário.

5. Canal de contato

Use o canal de atendimento informado pelo responsável local do aplicativo.
