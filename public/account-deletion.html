Solicitação de Exclusão de Conta

Esta página explica como solicitar a exclusão da conta e dos dados associados ao aplicativo Territórios.

1. Como solicitar a exclusão

  1. Entre em contato com o administrador ou responsável local pelo uso do aplicativo.
  2. Informe o e-mail usado para entrar no aplicativo.
  3. Solicite expressamente a exclusão da conta e dos dados vinculados ao seu cadastro.
  4. Se necessário, o responsável pode pedir uma confirmação mínima para validar a titularidade da conta.

2. Dados normalmente excluídos ou desvinculados

  • Cadastro do usuário no sistema, incluindo nome, e-mail, função de acesso e WhatsApp.
  • Token de notificações push associado ao cadastro.
  • Permissões de acesso vinculadas ao usuário dentro do aplicativo.

3. Dados que podem ser mantidos por período adicional

Alguns registros podem precisar ser mantidos pelo período mínimo necessário para segurança, prevenção a fraude, auditoria, integridade operacional, resolução de incidentes, cumprimento de obrigação aplicável ou preservação de histórico administrativo do sistema.

  • Registros históricos vinculados a operações administrativas já realizadas.
  • Logs técnicos e registros mínimos de segurança.
  • Informações que precisem ser preservadas por obrigação legítima aplicável.

4. Prazo de retenção adicional

Quando houver necessidade de retenção adicional, registros mínimos de segurança e dados operacionais estritamente necessários podem ser mantidos por até 90 dias. Se existir obrigação legal, necessidade de auditoria ou justificativa legítima de segurança, esse prazo poderá ser maior pelo tempo estritamente necessário.

5. Solicitação de exclusão de dados sem excluir a conta

Se o usuário quiser solicitar a exclusão de alguns dados sem excluir a conta, deve usar a página específica de exclusão de dados, com análise caso a caso conforme viabilidade técnica e necessidade administrativa.

6. Canal de contato

Use o canal de atendimento informado pelo responsável local do aplicativo para enviar a solicitação.
